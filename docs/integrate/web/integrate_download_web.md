# 通过npm工具进行下载

[点击这里](https://www.npmjs.com/package/open-im-sdk)前往npm package页面

- 进入项目命令行运行如下命令

  ```bash
  npm install open-im-sdk
  ```

  

# 通过CDN引入

- 在需要引入的html文件中通过script标签引入

  ```html
  <script src="https://cdn.jsdelivr.net/npm/open-im-sdk"></script>
  ```
